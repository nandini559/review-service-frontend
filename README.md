# Review Service Frontend 💅

## 🚀 Project Description

This is a frontend application for a review service system where users can:

- Create reviews
- View products
- Manage user accounts

---

## 🛠 Tech Stack

- React ⚛️
- TypeScript 🟦
- Tailwind CSS 🎨
- Vite ⚡

---

## 📦 Installation

```bash
git clone https://github.com/YOUR_USERNAME/review-service-frontend.git
cd review-service-frontend
npm install
npm run dev
```

Folder Structure
src/
├── components/
├── pages/
├── api/
├── App.tsx

Running the App

Open:

http://localhost:5173
